Sobre o Projeto

Este sistema foi desenvolvido utilizando Spring Boot no backend e Angular no frontend, com o objetivo de cumprir o desafio de desenvolvedor full stack Accenture.

O sistema permite o cadastro completo de empresas (pessoas jurídicas) e fornecedores (pessoas físicas ou jurídicas), oferecendo funcionalidades de criação, edição, exclusão e associação entre entidades, além de utilizar consulta via API por CEP.

Funcionalidades

  • Cadastro completo de empresas
  • Cadastro de fornecedores (PF e PJ)
  • Validação automática de documentos (CPF, CNPJ)
  • Consulta automática de endereços via CEP
  • Gerenciamento de associações

Tecnologias Utilizadas

Angular
Spring Boot
H2 DATABASE
Bootstrap
